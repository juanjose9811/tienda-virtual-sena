# Tienda Virtual - Inventario de Lavadoras

Proyecto desarrollado como evidencia para el SENA.

Tecnologías utilizadas:

- Flutter (Frontend)
- PHP (API REST)
- MySQL (Base de datos)
- XAMPP (Servidor local)
- InfinityFree (Hosting)

Funciones del sistema:

- Visualización del inventario
- Registro de lavadoras
- Actualización de productos
- Conexión entre aplicación móvil y API
